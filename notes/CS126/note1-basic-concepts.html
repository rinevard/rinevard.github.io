基本概念

首先，我们要指出随机变量(random variable) 和 其分布(distribution) 的区别。前者是一个从 $\Omega$ 到 $\mathcal{X}$ 的函数，后者是一个从 σ-field $\Sigma$ 到 $[0,1]$ 的概率测度。

  1. 离散随机变量具有 Probability mass function (pmf) - 概率质量函数，其定义为： $$p_X(x) = \mathbb{P}(X = x)$$
  2. 离散随机变量具有 Cumulative mass function (cmf) - 累积质量函数，其定义为： $$\mathbb{P}(X \leq x) = \sum_{n=-\infty}^{x} p_X(n)$$
  3. 连续随机变量具有 Cumulative distribution function (cdf) - 累积分布函数，其定义为： $$F_X(x) := \mathbb{P}(X \leq x) = \mathbb{P}(X \in (-\infty, x])$$
  4. Complementary cumulative distribution function (ccdf) - 互补累积分布函数和 Survivor function - 生存函数是同一个东西，其定义为： $$\bar{F}_X(x) := \mathbb{P}(X > x) = 1 - \mathbb{P}(X \leq x)$$
  5. Inverse distribution function - 反分布函数和 Percentile-point function (ppf) - 百分位点函数也是同一个东西，其定义为 $$F_X^{-1} : [0, 1] \to \mathcal{X}$$ $$F_X^{-1}(p) := \min \{x \in \mathbb{R} : F_X(x) \geq p\}$$
  6. 对独立同分布的随机变量$X_1, ..., X_n$，其 kth Order statistic - 第 k 个顺序统计量 被定义为这些随机变量中第 k 小的数值，记作 $X_{(k)}$，其 cdf(累积分布函数) 为： $$F_{X_{(k)}}(x) = \sum_{m=k}^{n} \binom{n}{m} [F_X(x)]^m \cdot [1 - F_X(x)]^{n-m}$$ $$= \sum_{m=k}^{n} \binom{n}{m} [\mathbb{P}(X \leq x)]^m \cdot [\mathbb{P}(X > x)]^{n-m}.$$

    7. 特别地，对 $X_{\min} = X_{(1)}$ 和 $X_{\max} = X_{(n)}$，有 $F_{X_{\min}}(x) = 1 - [1 - F_X(x)]^n$ 和 $F_{X_{\max}}(x) = [F_X(x)]^n$.

  7. 对两个相互独立的随机变量 $X$ 和 $Y$，定义它们各自分布的 Convolution - 卷积 为它们的和 $X+Y$ 的概率分布： $$\mathbb{P}(X + Y = n) = \sum_{k=-\infty}^{\infty} \mathbb{P}(X = k) \cdot \mathbb{P}(Y = n - k)$$ $$\text{if } X \text{ and } Y \text{ are discrete; if they are continuous, then}$$ $$f_{X+Y}(z) = (f_X * f_Y)(z) := \int_{-\infty}^{\infty} f_X(x) \cdot f_Y(z - x) \, dx.$$

    8. 这里的符号 $*$ 就是表示对两个函数做卷积的符号。
